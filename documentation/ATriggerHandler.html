    ATriggerHandler

    This class serves as a base implementation for trigger handler logic, providing default behavior for the various trigger events (before insert, update, delete, etc.). It implements the ITriggerHandler interface and extends ADMLDataProvider to process trigger actions. Subclasses should override specific methods to implement custom behavior.
